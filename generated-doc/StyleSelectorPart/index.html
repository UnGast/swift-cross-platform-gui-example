selects(_:​) 

public func selects(_ widget: Widget) -> Bool

Returns

true if type or typeName are either not set or match the widget and classes, pseudoClasses fully or partially match the given widget. Whereby the Widget can contain elements that are not in the selector, but the selector cannot contain elements that are not in the Widget. An empty part always selects any widget (returns true). Otherwise returns false.
