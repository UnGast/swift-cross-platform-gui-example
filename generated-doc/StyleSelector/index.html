selects(_:​) 

public func selects(_ widget: Widget) -> Bool

Checks whether the selector parts match starting from the last selector part and the given widget and going up the Widget tree checking the widgets parents. Selector parts that have .extendsParents == true are merged with their parents. Whether the first selector part is an extension on it's parent (and therefore the whole selector is) or not is ignored. Logic should be applied before calling this function to make sure that in case the StyleSelector is an extension on it's parent, the parent StyleSelectors that are managed by outside logic are checked as well.
